แบบฝึกปฏิบัติ: HTML Formatting

เรียนรู้การจัดรูปแบบข้อความด้วยแท็ก HTML พร้อมตัวอย่างและการตกแต่งด้วย CSS

📄 ตัวอย่างผลลัพธ์
B: ข้อความตัวหนา
STRONG: ข้อความสำคัญ
I: ข้อความตัวเอียง
EM: ข้อความเน้นเสียง
U: ข้อความขีดเส้นใต้
MARK: ข้อความที่ถูกเน้น
SMALL: ข้อความขนาดเล็ก
DEL: ข้อความที่ถูกลบ
SUP: x2
SUB: H2O
💻 ตัวอย่างโค้ด HTML 
<b>ข้อความตัวหนา</b>
<strong>ข้อความสำคัญ</strong>
<i>ข้อความตัวเอียง</i>
<em>ข้อความเน้นเสียง</em>
<u>ข้อความขีดเส้นใต้</u>
<mark>ข้อความที่ถูกเน้น</mark>
<small>ข้อความขนาดเล็ก</small>
<del>ข้อความที่ถูกลบ</del>
x<sup>2</sup>
H<sub>2</sub>O
🎯 กิจกรรมฝึกปฏิบัติ
  • ✓ ให้นักศึกษาสร้างหน้าเว็บ 1 หน้า
  • ✓ ใช้แท็ก formatting อย่างน้อย 6 รูปแบบ
  • ✓ ทดลองเปรียบเทียบ <b> กับ <strong> และ <i> กับ <em>
  • ✓ ปรับสี ขนาด และพื้นหลังของข้อความด้วย CSS
  • ✓ บันทึกไฟล์ชื่อ formatting.html
💡

เกร็ดความรู้

แท็กจัดรูปแบบข้อความช่วยให้เนื้อหาสื่อสารชัดเจนและมีความหมายมากขึ้น เช่น <strong> ใช้สำหรับข้อความสำคัญ <em> ใช้เน้นความสำคัญหรืออารมณ์ และ <mark> ใช้เน้นข้อความที่ต้องการให้โดดเด่น แท็กเหล่านี้ช่วยเพิ่มความหมายและช่วยจัดระเบียบเนื้อหาให้ผู้ใช้งานเข้าใจง่ายขึ้น
